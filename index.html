Overpass turbo (overpass-turbo.eu) é uma ferramenta de mineração de dados baseada na web para OpenStreetMap. 

Ela executa qualquer tipo de consulta da API Overpass e mostra os resultados em um mapa interativo. Usá-la pode se tornar técnica bem rápido: iniciantes podem achar o "assistente" um bom lugar para começar.

Instâncias

Versão estável: https://overpass-turbo.eu
Versão mais recente: https://tyrasd.github.io/overpass-turbo
Edição suíça: https://overpass-turbo.osm.ch

Executando consultas

Coloque sua consulta da API Overpass  no editor, clique no botão Executar e surpreenda-se com a qualidade dos dados OSM exibidos ;)

O overpass turbo exibe o máximo de dados possível (veja a chave do mapa abaixo). Ao clicar em um objeto, um pop-up mostra todas as informações do nó, caminho ou relação selecionado: tipo e id, tags, coordenadas, associações de relacionamento e metadados são mostrados, se disponíveis.

Observe que você precisa de um navegador da web relativamente recente para usar o overpass turbo. Opera, Chrome e Firefox foram testados e funcionam.

A API Overpass pode ser uma ótima ferramenta para mapeamento, pois é muito poderosa na filtragem de dados OSM.
Com o overpass turbo, há uma maneira fácil de executar rapidamente qualquer consulta Overpass e inspecionar os resultados de uma maneira amigável.





Aqui estão algumas ideias em que usar o "overpass-turbo" pode ser uma ferramenta útil durante o mapeamento:

Procurando por erros de ortografia (raros) ou quebras com convenções de nomenclatura em uma grande área.
Procurando por POIs especiais que não são desenhados no mapa.
Inspecionando POIs (por exemplo, nós de lugar) que são distribuídos uniformemente em grandes áreas.
Mostrando recursos espacialmente grandes (limites, rios, rodovias completas) e carregando-os diretamente em um editor.
quando você só precisa de uma parte filtrada dos dados OSM.
Para o desenvolvedor
O Overpass turbo também pode ser uma ferramenta para desenvolvedores:
Teste e desenvolvimento de consultas de API Overpass mais ou menos complicadas.
Conversão de dados OSM para o formato de dados geoJSON.
Criação de mock-ups de mapas clicáveis ​​ou estáticos destacando recursos OSM selecionados.
Este projeto é mantido por Martin Raifer. O código-fonte é encontrado no github GitHub. Um grande agradecimento a Roland que criou a API Overpass, sem a qual esta ferramenta não existiria.

Fonte: https://wiki.openstreetmap.org/wiki/Overpass_turbo
