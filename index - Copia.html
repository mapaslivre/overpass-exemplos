Overpass turbo (overpass-turbo.eu) é uma ferramenta de mineração de dados baseada na web para OpenStreetMap.
Ela executa qualquer tipo de consulta da API Overpass e mostra os resultados em um mapa interativo.
Usá-la pode se tornar técnica bem rápido: iniciantes podem achar o "assistente" um bom lugar para começar.
Este projeto é mantido por Martin Raifer. O código-fonte é encontrado no github.
 Um grande agradecimento a Roland que criou a API Overpass, sem a qual esta ferramenta não existiria.
